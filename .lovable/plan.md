

## Professional Multi-Sheet Estimation Report

The Excel and PDF exports will be restructured into a professional multi-sheet estimation file format with cover page, detailed bar list, size summary, bending schedule, and grand totals.

---

### Current State

The export currently produces:
- **Excel**: 5 sheets (Summary, Elements Detail, Size Breakdown, Bar List, Bending Schedule) with basic data columns
- **PDF**: Simple HTML with summary numbers, elements table, and size breakdown

### Target State

A professional estimation report matching industry standards:

---

### Excel Export (6 Sheets)

**Sheet 1 -- Cover Page**
- Company branding row: "Rebar Estimator Pro"
- Project name, client name, project type
- Date generated
- Rebar coating type
- Scope items included
- Project-specific deviations/notes
- Grand total weight (lbs and tons)
- Element count summary (Ready / Flagged / Excluded)

**Sheet 2 -- Bar List (main takeoff)**
- Grouped by element type (FOOTING, COLUMN, BEAM, etc.)
- Columns: Element ID, Bar Mark, Size, Shape Code, Qty, Length (ft), Unit Weight (lb/ft), Total Weight (lbs)
- Subtotal row after each element type group
- Grand total row at the bottom
- Column widths auto-fitted, headers styled with background color

**Sheet 3 -- Size Summary**
- Columns: Rebar Size, Total Weight (lbs), Percentage (%)
- Total row at bottom
- Sorted by size (#3 through #18)

**Sheet 4 -- Bending Schedule**
- Only bent bars (non-straight, non-closed)
- Columns: Element ID, Bar Mark, Size, Shape Code, Qty, Length (ft), Weight (lbs)
- Grouped by shape code

**Sheet 5 -- Elements Detail**
- Per-element summary: Element ID, Type, Status, Confidence (%), Total Weight (lbs)
- Color coding: green for READY, amber for FLAGGED, red for BLOCKED

**Sheet 6 -- Notes**
- Project deviations text
- Scope items selected
- Calculation mode used
- Disclaimer/footer text

All sheets will have:
- Styled headers (bold, colored background)
- Auto-fitted column widths
- Number formatting (commas for thousands, 1 decimal for weights)
- Frozen header rows

---

### PDF Export (Multi-Section)

The PDF will be restructured with proper page sections:

1. **Header**: "Rebar Estimation Report" + project name + client + date
2. **Project Summary**: Total weight, tons, element counts, mode, coating type
3. **Bar List Table**: Grouped by element type with subtotals (same as Excel Sheet 2)
4. **Size Summary Table**: Size breakdown with percentages
5. **Bending Schedule**: Bent bars only (if any)
6. **Footer**: "Generated by Rebar Estimator Pro" + timestamp

Better print CSS with page breaks between sections.

- Use `!merges` for title rows spanning multiple columns
- Group bar list entries by `element_type` and insert subtotal rows
- Add `REBAR_WEIGHT` lookup table in the export for unit weight column
- PDF uses CSS `@page` rules and `page-break-before` for multi-page layout
